사용자가 입력한 값들을 서버로 넘기면서 요청하는 역할을 수행

form action="" method=""

*action : 해당 form에 사용자가 입력한 값들을 전달하면서 요청을 보낼 서버의 경로를 제시
*method : 요청 전송방식을 지정하는 속성 (get(기본값)/post) > get : 요청시 사용자가 입력한 값들이 url에 그대로 노출이된다. > post : 요청시 사용자가 입력한 값들이 url에 노출되지 않는 방식 form내에 submit버튼 클릭시 action에 지정한 서버로 사용자가 입력한 값들이
넘어가면서 요청됨(key=vlaue세트로 전달)

selectbox

무조건 하나를 선택해야할 때

radio button

selectbox처럼 여러개 중에 하나를 선택할 때 주로 사용함
name속성을 통해서 여러개의 라디오버튼을 하나의 선택지로 묶을 수 있음
checked속성을 넣은 라디오버튼이 기본값으로 설정됨
id - for 속성을 통해서 input과 label태그를 묶어줄 수 있음
서버에 실제로 전달될 때 key -> name속성값, vlaue -> value속성값으로 전달

checkbox

여러개를 선택할 때 사용

다중 선택할 시 주로 사용함
name속성을 통해서 여러개의 체크박스를 하나의 선택지로 묶을 수 있음
checked속성을 넣은 체크박스는 체크가 된상태가 기본이 됨
id - for 속성을 통해서 input과 label태그를 묶어줄 수 있음
서버에 실제로 전달될 때 key -> name속성값, vlaue -> value속성값으로 전달
