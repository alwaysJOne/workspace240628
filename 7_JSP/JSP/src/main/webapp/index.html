*JSP

JSP란, HTML내에서 자바코드를 쓸 수 있는 "자바언어"로
Servlet에서 응답화면(html)을 구현했던 복잡함을 간단하게 하는 기능
=> 즉, JSP의 가장 큰 장점은 Servlet에서 비즈니스로직에만 집중하고
프리젠테이션로직(응답화면만들기)은 JSP에서 집중하게금 분리할 수 있는 것

JSP Element표현법

1. JSP스크립트 원소

JSP페이지에서 자바코드를 기술할 수 있게 해주는 기능

  1. 선언문 : <%! 자바코드 %>
    멤버변수와 메소드를 선언하기 위해서 사용
  2. 스크립틀릿 : <% 자바코드 %>
    JSP에서 자바코드를 기술하기위한 제일 기본적인 공간
  3. 표현식(출력식) : <%=자바코드 %>
    자바에서의 값을 화면상에 출력하기위해 사용
view detail

2. 지시어 <% page | include | taglib 속성 = "속성값" 속성 = "속성값" %>

JSP페이지 전체에 영향을 미치는 정보를 기술할 때 사용함

  1. page 지시어 : 현재 jsp페이지를 처리하는데 필요한 각종 속성들을 기술하는 부분
    • language : 사용할 언어 유형을 지정
    • contentType : 해당 페이지의 형식, 문자셋 지정
    • pageEncoding : JSP페이지내에 기록된 자바코드의 인코딩방식
    • import : 자바의 impoprt와 같은 의미
    • errorPage : 해당 jsp상에 요류 발생했을 경우 보여줄 여러 에러페이지를 기술
    view detail
  2. include 지시어 : 현재 jsp상에 또다른 jsp를 포함시키고자 할 때 사용하는 지시어 view detail
  3. taglib 지시어 : jsp기능을 좀더 확장시킬 수 있는 추가적인 라이브러리 등록하는 문구
